Color Palette

Primary Colors

Teal -
HEX: #48cae4 | RGB: 72, 202, 228 | CMYK: 68, 11, 0, 11

Teal was chosen as the primary accent color because it evokes a sense of modernity, technology, and exploration—perfectly aligned with the space and futuristic theme of the site. Teal stands out well against dark backgrounds, creating a visually striking effect, which is essential for grabbing attention in key areas such as headers, buttons, and interactive elements.

Dark Background -
HEX: #121212 | RGB: 18, 18, 18 | CMYK: 0, 0, 0, 93

A dark background was chosen to mimic the vastness and mystery of space, creating a sleek, immersive experience for users. This also helps the teal accent and off-white text stand out, enhancing readability and user focus. The dark background is ideal for creating contrast, allowing the content and visuals to pop.

Off-White Text -
HEX: #eeeeee | RGB: 238, 238, 238 | CMYK: 0, 0, 0, 7

Off-white was selected for text because pure white can be too stark and create eye strain against a dark background. A softer off-white (#eeeeee) balances readability with comfort, ensuring that users can consume content easily without feeling overwhelmed by harsh contrasts.

Secondary Colors

Soft Grey -
HEX: #cccccc | RGB: 204, 204, 204 | CMYK: 0, 0, 0, 20

Soft grey is used for secondary text and UI elements. It helps to create a visual hierarchy, subtly distinguishing less important content without causing a jarring break in the color scheme. The color is light enough to maintain legibility but subdued to not compete with the primary white and teal elements.

Yellow -
#FFD700 | RGB: 255, 215, 0 | CMYK: 100, 0, 0, 0

The Yellow is used sparingly for hover effects and icons. It adds a modern, almost futuristic glow that resonates with the site’s space theme. The neon color is bright enough to capture attention, but used judiciously to avoid overwhelming users.

Navigational Elements

Sticky Navigation Bar

Purpose-
The sticky navigation bar keeps the main menu accessible at all times, allowing users to easily switch between sections of the site .

This improves user experience by reducing the need for excessive scrolling and ensuring users can explore all parts of the site without interruption, particularly helpful for large content-rich sections.

Back to Top Button

Purpose-
The back-to-top button lets users quickly navigate to the top of long-scrolling pages.

It aids navigation, particularly on pages with long content, improving user efficiency and reducing frustration. The design includes rounded edges, echoing the space theme, with a glow effect for visual feedback.

Breadcrumb Navigation

Purpose-
Breadcrumbs help users keep track of their location on the site and easily return to previous pages.

This feature improves site wayfinding, especially in the blog section. It helps users better understand the structure of the site and navigate efficiently, supporting an exploration-driven experience.

Next/Previous Blog Navigation

Purpose-
Provides seamless navigation between blog posts.

This encourages users to explore more content without having to return to the blog index. The buttons feature rounded corners and icons to match the space theme, ensuring consistency.

Tabs for Section Navigation

Purpose-
Tabs organize content into manageable sections, making it easy for users to navigate large pages.

This is especially useful for pages with multiple sections, such as the style guide, theory, and wireframes, allowing users to focus on one section at a time. Active tabs are highlighted to indicate user position.

Interactivity Elements

Buttons with Rounded Corners

buttton preview

Purpose-
Buttons have rounded corners to reflect the roundness of planets, aligning with the site’s space theme (hover over them above to see).

Rounded buttons provide a softer, more approachable interface. The glow effect on hover enhances user engagement, giving a responsive feel while visually reinforcing the futuristic aesthetic.

Icons Paired with Buttons

Purpose-
Icons are paired with buttons to provide clear visual cues and enhance accessibility.

Icons make buttons more intuitive and accessible, improving usability by providing additional visual context. This aligns with the goal of making the site easy to navigate, even for users with varying technical skills.
